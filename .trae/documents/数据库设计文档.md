# 数据库设计文档

## 概念模型
- Category：类别
- Album：相册
- Photo：照片
- Reservation：预约（预留）
- Package：套餐（预留）

## 逻辑模型
### Category
- id: string
- name: string
- slug: string
- coverImage: string

### Album
- id: string
- title: string
- categoryId: string
- description: string
- coverImage: string
- shootDate: date

### Photo
- id: string
- albumId: string
- src: string
- width: int
- height: int
- lqip: string
- tags: string[]
- dominantColor: string

### Reservation（预留）
- id: string
- clientName: string
- phone: string
- date: datetime
- packageId: string
- status: enum
- note: string

### Package（预留）
- id: string
- name: string
- price: number
- features: string[]

## 说明
- 初期可采用本地 JSON 存储，后续接入 SQLite/Postgres 或托管服务
- 图片元数据可在构建期生成并写入 JSON
