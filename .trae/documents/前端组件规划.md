# 前端组件规划

## 页面
- Home：主视觉、精选合集、快速入口
- Gallery：分类筛选、瀑布流/网格、分页
- PhotoDetail：大图、EXIF/标签、相关作品
- About/Contact：品牌介绍与联系信息

## 组件
- GalleryGrid：响应式网格/瀑布流
- LazyImage：懒加载 + LQIP +交错淡入
- Lightbox：全屏查看、键盘/手势导航
- Carousel：轮播、自动/手动切换、缩略图
- FilterBar：筛选分类、标签与排序
- CategoryTabs：类别导航与状态保持
- Breadcrumbs：层级导航

## 技术策略
- Astro 模板 + React 岛组件
- 交互组件按需加载与视口内激活
- 路由状态通过 URL 参数保持
