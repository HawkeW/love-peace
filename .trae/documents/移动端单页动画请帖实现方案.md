## 动效与交互设计
- 进入动效：图片先 0→1 的淡入，文本稍后淡入 + 微量上移（translateY）。
- 切换动效：上一屏淡出，下一屏淡入；避免复杂滑动导致眩晕。
- 交互手势：点击整屏或上滑进入下一屏；底部按钮备选（“下一页”）。
- 可访问性：ARIA 标签与按钮焦点可达；支持“减少动态”系统设置降级为无动画。

## 技术实现（Astro）
- 新组件 SlidePage.astro：
  - props：{ src, title, copy, index, total }
  - 使用 IntersectionObserver 添加 .visible class 触发 CSS 动效。
  - 文案层（overlay）采用安全边距与响应排版，适配暗/亮模式。
- 页面 invitation-story.astro：
  - 从固定数组加载 slides（精选素材 + 文案），每屏一个 SlidePage。
  - 全屏纵向布局；点击或滚动触发下一屏，附进度指示。
- 文案来源：优先人工精选；可选从 [descs](file:///d:/Oncew/shae-hughe/love-peace/public/descs) 读取首行为标题、首段为摘要填充。

## 移动端适配与性能
- 视口与排版：最大宽度 480–640px，自适应字号；安全区域（刘海屏）。
- 图片性能：按屏预加载下一张；使用 srcset（720/1080）与 lazy 加载。
- 降级策略：prefers-reduced-motion 时禁用动画；iOS Safari 使用 transform+opacity，避免滤镜性能坑。
